Miscellaneous -- Compiled for v1.2.5

Shell Completions:
Swordfish officially supports completions for Bash, Zsh, and Fish. Generate them
with "--completions <shell>". Output goes to stdout by default, or to a file if provided.

  --completions <shell> [file]   Generate completions for fish, bash, or zsh

Examples:
  swordfish --completions fish                  Print fish completions to stdout
  swordfish --completions bash ~/.bash_complete Write bash completions to a file

Version:
  --version              Show the current installed version of Swordfish

Man Pages:
Man pages are generated automatically on every build and written to docs/swordfish.1.
You can also generate them manually and write to a custom path:

  --man [file]           Generate the man page, optionally writing to <file>

The man page is built dynamically from all help pages combined and formatted.
