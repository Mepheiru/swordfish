General Options -- Basic Usage for Swordfish -- Compiled for v1.2.5

All flags can be grouped together. For example: "-K -S" can be written as "-KS".
Below are the commonly used options for controlling how Swordfish finds and interacts with processes.

  -h, --help             Show help message
  -v                     Increase verbosity level up to -vvv for maximum verbosity
  -k                     Send SIGTERM to matching processes (graceful shutdown)
  -K                     Send SIGKILL to matching processes (forceful shutdown)
  -S                     Select which processes to act on (interactive prompt)
  -y                     Auto-confirm; skip prompts and confirmation
  -p                     Print raw PIDs only

Supplying a pattern without any signal flags will enter "Static Mode" where Swordfish will
find and display matching processes without sending any signals. Useful for checking
if something is running before you decide to kill it.

By default, Swordfish asks for confirmation before sending signals. Use "-y" to skip
this and execute immediately.

Run 'swordfish --help <category>' for more detail on a specific topic.
