Arguments -- Full Argument Reference -- Compiled for v1.3.0

Operations (mutually exclusive, at most one per invocation):
  -k                     Send a signal to matching processes (default: SIGTERM)
                         Signal embeds directly after -k: -k9, -kHUP, -k15
  -S                     Interactively select which processes to act on
  -W                     Watch matching processes live

Modifiers (stackable, combine freely with operations):
  -x                     Exact match process names (default: substring match)
  -y                     Auto-confirm; skip all prompts
  -v                     Increase verbosity (up to -vvv)
  -t                     Always act on only the top matched process
  -p                     Print raw PIDs only
  -n                     Dry run — show what would happen without doing it
  -w                     Wait for the process to die after sending a signal
  -r                     Hide processes owned by root

Filtering (long opts):
  --user <user>          Filter processes by username
  --exclude <pattern>    Exclude processes matching <pattern>
  --sort <ram|age>       Sort process list by RAM usage or age
  --parent <pid>         Match only children of the given parent PID
  --session <sid>        Match processes by session ID
  --pidfile <file>       Read target PID from file

Behavior (long opts):
  --retry <seconds>      Retry every <seconds> if no match is found
  --timeout <seconds>    Escalate to SIGKILL after <seconds> if process does not die
  --format <string>      Custom output format string

Hooks:
  --pre-hook <script>    Run <script> before sending signals
  --post-hook <script>   Run <script> after sending signals

Misc:
  --completions <shell> [file]   Generate shell completions (fish, bash, zsh)
  --man [file]           Generate man page, optionally writing to <file>
  --version              Show installed version
  --help [category]      Show help, optionally for a specific category
